# Treasure Mapper

Category: Casual Games

A map-reading puzzle game where every adventure starts with a hand of clues. A procedurally generated 2D top-down world is laid out before you — trees, streams, rocks, and clearings — along with a treasure map that reads something like: "Starting at the big oak, walk 10 paces east, then 12 paces south. X marks the spot." Select the cell where you think the treasure is buried and tap Dig Here. You have three tries — if you're stumped, the game will show you where it was hiding. Then a fresh world and a fresh map await.
